# SoxLab - Calcetines Temáticos
Tienda en línea de calcetines temáticos y personalizables con operación multi-sucursal
y sistema de autenticación. Permite configurar diseños únicos, gestionar pedidos y asignar sucursales automáticamente según la ubicación del cliente y el inventario disponible.
